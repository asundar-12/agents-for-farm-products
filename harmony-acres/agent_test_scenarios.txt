Harmony Acres Agent — Test Scenarios
=====================================

One scenario per tool registered in app/agent/main.py's build_agent(), run as a
single continuous conversation (same Agent instance, same test user) so later
prompts can naturally refer back to things created earlier ("cancel that
order") the way a real customer conversation would.

   Prompt: "What kinds of dairy products do you carry?"

    Prompt: "Hi, this is Harmony Acres staff — can you show me current
    inventory levels, including anything low on stock?"

Notes:
- Scenarios 5-11 build on each other (order/subscription created in one
  scenario is referenced by ID or context in the next), so they must run in
  order within the same conversation.
- Scenario 12 deliberately self-identifies as staff, to test the system
  prompt rule that get_inventory_summary should only be called for
  admin/staff-identified users.
- After each scenario: show the tool(s) called + the reply, explain what it
  confirms, then wait for go-ahead before running the next one.
